## Deployment to GitHub Pages

### Option 1: Main Repository Website (username.github.io)

1. Create a new repository named `frankfurtmacmoses.github.io`
2. Copy `index.html` and `about.html` to the repository root
3. Commit and push
4. Your site will be available at: `https://frankfurtmacmoses.github.io`

### Option 2: Project Repository Website

1. Push these files to any GitHub repository
2. Go to repository Settings â†’ Pages
3. Select source: Deploy from a branch
4. Select branch: main (or master)
5. Select folder: / (root) or /frankpersonal
6. Your site will be available at: `https://frankfurtmacmoses.github.io/repository-name`
